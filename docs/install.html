Install

Install from conda-forge

To install nsbmm from Anaconda use the following command

conda install nsbm -c conda-forge

Build from source

Or you can build it from source

Dependencies:

  • graph-tool > 2.40

  • numpy

  • matplotlib

  • cloudpickle

  • pandas

python3 -m pip install . -vv
